无闻-go语言编程基础笔记

一:go的环境变量与工作目录

GOPATH下需要建立3个目录

  • bin （存放编译后生成的可执行文件）
  • pkg （存放编译后生成的包文件）
  • src （存放项目的源码）

二:go常用命令

  • go get : 获取远程包（需要提前安装 git或hg）
  • go run : 直接运行程序
  • go build : 测试编译，检查是否有编译错误 （package main ）可执行入口
  • go fmt : 格式化源码
  • go install : 编译包文件并编译整个程序
  • go test : 运行测试文件 （**_test.go是测试文件）
  • go doc : 查看文档(本地官网 go -http=:8080 & 后台执行)

三：go注释

三: go程序结构

  • package main包含main函数

    • 一个可执行程序只有一个main包

  • import 导入非main的包

##　import 导入包格式

可见性规则

go语言：使用大小写来决定： 常量，变量，类型，接口，结构，函数。是否被外部包所调用。

函数名大写，即为：private

函数名小写，即为：public
